# 🏠 HouseRent - House Rental Management System

A full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application for managing house rental listings. The system enables property owners to list rental properties and tenants to search, view, and book properties online.

---

## 📌 Features

### Tenant
- User Registration & Login
- Browse available properties
- Search and filter properties
- View property details
- Book rental properties

### Property Owner
- Secure Login
- Add new properties
- Update property details
- Delete properties
- View booking requests

### Admin
- Manage users
- Manage properties
- Monitor bookings

---

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yogeswar1925-ship-it/HouseRent.git
```

### Go to Project

```bash
cd HouseRent
```

### Backend

```bash
cd "source code/Server"
npm install
npm start
```

### Frontend

```bash
cd "source code/Client"
npm install
npm run dev
```
