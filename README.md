# 🏠 HouseRent - House Rental Management System

A full-stack **MERN (MongoDB, Express.js, React.js, Node.js)** web application designed to simplify the process of renting and managing residential properties. The platform allows property owners to list and manage rental properties while enabling tenants to search, view, and book available houses through an intuitive and responsive interface.

---

## 📖 Project Overview

HouseRent is a modern web-based rental management system that connects property owners with tenants on a single platform. The application provides secure authentication, efficient property management, and a seamless booking process using RESTful APIs and MongoDB.

## ⚙️ Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/yogeswar1925-ship-it/HouseRent.git
```

### 2. Navigate to the Project Folder

```bash
cd HouseRent
```

### 3. Install Backend Dependencies

```bash
cd "source code/Server"
npm install
npm start
```

### 4. Install Frontend Dependencies

Open a new terminal and run:

```bash
cd "source code/Client"
npm install
npm run dev
```

## 🔐 Authentication

The application uses **JSON Web Token (JWT)** authentication to secure user sessions and protect API endpoints. Passwords are encrypted using **bcrypt.js** before being stored in the MongoDB database.
